# Tabela médicos, geração de dados fonte

Objetivo: gerar um arquivo `Medicos.xls` com dados de médicos de um sistema de saúde.
Gere registros de 20 médicos. Gere dados em formato bruto, sem formatação e sem nenhum tipo de processamento.


## Colunas

ID: numérico, incremental, ordenado
Nome: string
CPF: string
Data de nascimento: string (formato: 2026-05-29)
Sexo: string (M ou F)
